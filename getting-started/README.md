# Getting Started

Welcome to VMStation! This section will help you get up and running with your homelab Kubernetes environment.

## Quick Navigation

- [Quick Start Guide](quick-start.md) - Get started in minutes
- [Prerequisites](prerequisites.md) - What you need before starting
- [Installation Guide](installation.md) - Detailed installation steps
- [First Deployment](first-deployment.md) - Deploy your first workload

## Recommended Path

1. **Review Prerequisites** - Ensure you have the required hardware and software
2. **Follow Quick Start** - Get a basic cluster running
3. **Complete Installation** - Configure additional features
4. **Deploy First Application** - Validate your setup works

## What You'll Build

By the end of this guide, you'll have:

- A 3-node Kubernetes cluster
- Complete monitoring stack (Prometheus, Grafana, Loki)
- Automated power management (Wake-on-LAN, auto-sleep)
- Infrastructure services (NTP, Syslog)
- Ready-to-use dashboards
